# Cliente React - Panel C&C de Botnet
Cliente web desarrollado con React y Vite que simula un panel de Command & Control (C&C) para una botnet.

## Instalación

```bash
npm install
```

## Ejecución

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`

## Configuración
Para cambiar la URL del servidor de Flask, modifica la constante `SERVER_URL` en `src/App.jsx`.
